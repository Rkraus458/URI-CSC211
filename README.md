# CSC 211 — Computer Programming (OOP)
**University of Rhode Island**

> Problem specification, solution design, and algorithm development. Topics include data types,
> functions, iteration, recursion, object-oriented programming, built-in data structures, file
> operations, numerical and string-based operations.

**Language:** C++

### Assignment 5 — Cell Grid Simulation
A more complex assignment implementing a grid-based simulation with bitmap rendering.
- `cellgrid/cellgrid.cpp` / `cellgrid.h` — Grid data structure
- `imageBin/bitmap_image.hpp` — Bitmap image library
- `imageBin/bitmap_test.cpp` — Image generation tests
